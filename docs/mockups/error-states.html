Consistency plan. The light alert above is the same component as Company info's error box (T7.6) — extracted into a shared app-error-alert (inputs: title, message; optional (retry) output) under shared/ui/, alongside the existing Modal/Toast primitives. Invoices, Company info, and (next) Generate all render it, so every page's error looks and behaves identically: light surface, danger rail, alert glyph, readable text, solid Retry. The shimmer .sk primitive is likewise shared; each page composes its own skeleton shape (table rows here, key/value cards on Company info). On a load failure each page also enqueues the existing error Toast — unchanged.
